# Node.js + TypeScript API

Base inicial para una API en Node.js con TypeScript y Express.

## Requisitos

- Node.js 20+

## Instalacion

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Estructura

```text
src/
  config/
  controllers/
  routes/
  app.ts
  server.ts
```
