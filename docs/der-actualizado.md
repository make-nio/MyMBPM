# DER Actualizado

## Cambios Aplicados

- `PEDIDO_DETALLE` agrega `COSTO_UNITARIO`
- `PEDIDO` agrega `NUMERO_PEDIDO`
- `PEDIDO` agrega `SUBTOTAL`
- `ESTADO_STOCK` agrega `ID_USUARIO`
- `ITEM_CATALOGO` agrega `STOCK_MINIMO`
- se crea `ORDEN_PRODUCCION_CONSUMO`
- en `ITEM_CATALOGO_COMPONENTE` se reemplaza `ID_ITEM_CATALOGO_COMPONENTE_HIJO` por `ID_ITEM_CATALOGO_HIJO`

## Nota de Compatibilidad

El rename pedido para `ID_ITEM_CATALOGO_COMPONENTE_HIJO` no se aplico literalmente porque `ID_ITEM_CATALOGO_COMPONENTE` ya es la PK de la tabla. Para evitar colision y mantener claridad, se usa `ID_ITEM_CATALOGO_HIJO`.
